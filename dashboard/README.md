# Streamlit Dashboard

프로젝트 분석 산출물을 발표·포트폴리오용으로 탐색하는 대시보드입니다.

프로젝트 루트에서 실행합니다.

```powershell
uv run streamlit run dashboard\app.py
```

브라우저에서 `http://localhost:8501`을 엽니다.

대시보드는 `outputs/derived`, `outputs/tables`, `outputs/powerlifting_causal_analysis_results.xlsx`를 읽습니다. 메인 노트북을 먼저 끝까지 실행해야 합니다.
